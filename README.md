<h1>E-Shop E-Commerce Website</h1> 
<hr/>
E-Shop is an online e-commerce website that specializes in selling products for men, women, kids & electonic item like mobile, tv & laptop. This platform allows customers to browse and purchase a wide range of products, including cloths, shoes, watches, and more.

<h2>Getting Started</h2>
<hr/>
To access E-Shop's website, simply visit our homepage at www.eshop.com. Customers can browse and search for products by entering keywords in the search bar or by browsing different categories available on the website.

<h2>Features</h2>
<hr/>

User-friendly Interface
<br/>
Wide variety collection of Products
<br/>
Filter by Category, Brand Name
<br/>
Sort by Name, Price
<br/>
Set Limit Per Page
<br/>
Search for products by title, brand, or keyword
<br/>
Browse different categories of products
<br/>
View product details, including price, description, and reviews
<br/>
Add products to a shopping cart and checkout
<br/>
Option to create an account to track orders and save preferences
<br/>
Admin Page where you can add products, edit product details, see all the registered users, all the details in dashboard, all the product listings, all the orders & many more.
